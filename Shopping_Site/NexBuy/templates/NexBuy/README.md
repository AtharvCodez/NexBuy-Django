# NexBuy

**Oct 2025**

A futuristic multi-page e-commerce website built with a sleek dark theme, vibrant orange accents, and smooth GSAP animations — delivering a premium, interactive shopping experience.

---

## 🚀 Features

1. **Interactive Product Page** with category filters, price sliders, bank offers, and add-to-cart functionality  
2. **Smart Preloader Animation** for seamless page transitions  
3. **Customer Testimonial Section** with animated review graphs, star ratings, and feedback forms  
4. **AI Assistant Floating Button** concept for future conversational shopping features  
5. **Responsive Navigation Bar** and sticky bottom navbar for mobile devices  
6. **Smooth Scroll Animations** and glowing hover effects for enhanced engagement  
7. **About & Contact Sections** with modern layout and responsive design  
8. **Dynamic Pricing Section** with monthly and annual subscription toggles  
9. **Optimized Project Structure** for performance and reusability across pages  

---

## 🛠 Tech Stack

- HTML  
- CSS  
- JavaScript  
- GSAP (Animations)  
- Gemini Pro (used as an AI assistant for refining code & animations)
